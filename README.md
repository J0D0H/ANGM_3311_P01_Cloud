# ANGM_3311_P01_Cloud
Maya, PySide2, and Python cloud generator in maya

This will be a full weather generation tool. It will create a small scene that can create a cloud of various sizes, generate lightning from the cloud and show rain particles throughout the scene. 
This tool will have an integrated GUI that allows you to toggle any of the three objects on or off as desired. This GUI will also allow the user to increase or decrease the size of the cloud, the amount of rain, and how much lightning there is.
